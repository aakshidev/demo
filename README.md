# To-Do List App
Author- Aakshi
<hr>
A To-Do List app which would help you plan your day efficiently!
<hr> 
<b>How to use?</b>
<br>
- Choose the categorgy of list u want to create
<br>
- Enter the task you want to do in text box 
<br>
- Complete the task and cross it out by clicking on it
<hr>

Helps to keep a track of your productivity in a planned manner.
